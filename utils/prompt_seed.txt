Genera un unico prompt en ingles, listo para DALL-E, para crear UNA sola pieza publicitaria vertical de alto impacto para Facebook e Instagram de una empresa de tecnologia. El resultado debe verse como diseno grafico premium para redes sociales, no como fotografia real.

Datos de la empresa:
- Empresa: NoyeCode
- Slogan de referencia: "Tu Casa de Desarrollo de Software"

- Sitio web: noyecode.com

Objetivo de marketing:
- Vender un solo servicio por imagen.
- Priorizar claridad, atractivo visual y conversion.
- La promesa debe sentirse moderna, confiable y orientada a resultados.
- El anuncio debe captar atencion en menos de 2 segundos.

Servicio protagonista:
Elige SOLO UNO de estos servicios como enfoque principal:
- Desarrollo de software a la medida
- Automatizaciones empresariales
- Apps empresariales para campo (GPS, tracking, reportes)
- Software para negocios (CRM, ERP, inventario, ventas)

- Dashboards y analisis de datos en tiempo real
- Desarrollo Android y desktop

Enfoque visual obligatorio:
- Debe parecer un anuncio grafico para social media de empresa tech moderna.
- NO debe parecer foto realista.
- NO usar personas reales, manos, rostros, oficinas, escritorios ni escenas fotograficas.
- Usar fondo limpio con gradiente elegante y composicion minimalista.
- Incluir 1 o 2 dispositivos maximo (laptop, smartphone o tablet).

Composicion:

- Margenes amplios y jerarquia visual clara.
- La parte superior del 15% debe quedar totalmente vacia, como una barra plana de color solido oscuro #1a1a2e.

- Debe existir una zona clara para:
  titulo principal corto,
  subtitulo opcional muy breve,
  mockup del producto,
  CTA corto,
  web y telefono en pequeno.

Texto dentro de la imagen:
- Todo el texto visible dentro de la imagen debe estar en espanol.
- Usar muy poco texto para maximizar legibilidad.
- Titulo principal: maximo 6 palabras, directo y vendedor.
- Subtitulo opcional: maximo 8 palabras.
- CTA: maximo 3 palabras.
- Contacto al pie: "noyecode.com" y "+57 301 385 9952".
- NO escribir "NoyeCode", "Noye Code", "NOYECODE" ni ninguna variacion dentro de la imagen.
- NO pedir slogans largos ni bloques de texto.
- El texto debe verse nitido, bien espaciado y facil de leer.

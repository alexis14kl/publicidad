Genera un unico prompt en ingles, listo para DALL-E, para crear UNA sola pieza publicitaria vertical de alto impacto para Facebook e Instagram de una empresa de tecnologia. El resultado debe verse como diseno grafico premium para redes sociales, no como fotografia real.

Datos de la empresa:
- Empresa: NoyeCode
- Slogan de referencia: "Tu Casa de Desarrollo de Software"
- Telefono/WhatsApp: +57 301 385 9952
- Sitio web: noyecode.com

Objetivo de marketing:
- Vender un solo servicio por imagen con enfoque en OFERTAS y CONVERSION.
- La imagen debe funcionar como un anuncio de venta directa que motive al usuario a contactar.
- Incluir ganchos comerciales: precios desde, descuentos, cotizacion gratis, demo sin costo, cupos limitados.
- La promesa debe sentirse moderna, confiable y orientada a resultados.
- El anuncio debe captar atencion en menos de 2 segundos y generar accion inmediata.

Servicio protagonista:
Elige SOLO UNO de estos servicios como enfoque principal:
- Desarrollo de software a la medida
- Automatizaciones empresariales
- Apps empresariales para campo (GPS, tracking, reportes)
- Software para negocios (CRM, ERP, inventario, ventas)
- Apps no-code para automatizar procesos y dashboards
- Dashboards y analisis de datos en tiempo real
- Desarrollo Android y desktop
- Soporte tecnico informatico
- RPAs nativos
- Modernizacion de software legacy

Enfoque visual obligatorio:
- Debe parecer un anuncio grafico de oferta para social media de empresa tech moderna.
- NO debe parecer foto realista.
- NO usar personas reales, manos, rostros, oficinas, escritorios ni escenas fotograficas.
- Usar fondo limpio con gradiente elegante y composicion minimalista.
- Incluir 1 o 2 dispositivos maximo (laptop, smartphone o tablet).
- En la pantalla de los dispositivos debe verse una interfaz SaaS creible, moderna y ordenada con dashboards, graficos, metricas, tablas o mapas segun el servicio.
- La interfaz en pantalla debe apoyar el mensaje comercial, no robar protagonismo al titulo.

Composicion:
- Formato vertical 4:5.
- Margenes amplios y jerarquia visual clara.
- La parte superior del 12% de la imagen debe ser un area oscura que se integre suavemente con el fondo del diseno (NO una barra cortada ni un parche separado). Usar el mismo tono del gradiente superior del fondo para que fluya naturalmente. Esta zona se usara para superponer el logo despues, asi que no poner nada ahi.
- Debe existir una zona clara para:
  titulo principal corto con oferta o gancho comercial,
  subtitulo con beneficio concreto,
  mockup del producto,
  CTA urgente y directo,
  web y telefono en pequeno.

Texto dentro de la imagen:
- Todo el texto visible dentro de la imagen debe estar en espanol.
- Usar muy poco texto para maximizar legibilidad.
- Titulo principal: maximo 6 palabras, directo y vendedor, con gancho de oferta.
- Subtitulo opcional: maximo 8 palabras con beneficio concreto.
- CTA: maximo 4 palabras, urgente y accionable.
- Contacto al pie: "noyecode.com" y "+57 301 385 9952".
- NO escribir "NoyeCode", "Noye Code", "NOYECODE" ni ninguna variacion dentro de la imagen.
- NO pedir slogans largos ni bloques de texto.
- El texto debe verse nitido, bien espaciado y facil de leer.
